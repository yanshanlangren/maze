# 3D迷宫逃脱 - 多人合作版

基于 Three.js 和 Python aiohttp WebSocket 的可协同多人迷宫逃脱游戏。

## 游戏玩法

- 2 名玩家可进入同一个房间进行合作
- 一人到达终点，全队共同进入下一关
- 共有 16 个关卡，难度递增，僵尸越来越多
- 控制方式：WASD 移动，空格跳跃，鼠标控制视角，左键攻击

## 特性

- ✅ 随机生成迷宫（使用种子保证房间内所有玩家迷宫一致）
- ✅ 僵尸AI追逐玩家，可被击晕
- ✅ 陷阱伤害，3条生命
- ✅ 小地图显示玩家、僵尸、陷阱位置
- ✅ 玩家位置实时同步
- ✅ 自动重连机制
- ✅ 超时玩家自动清理

## 运行方法

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动服务器

```bash
python server.py
```

### 3. 访问游戏

打开浏览器访问 `http://localhost:5000`

多人游戏可在不同设备（同一局域网下）访问服务器IP进行联机。

## 优化改进

本版本优化了以下内容：

1. **服务器端**
   - 添加了结构化日志，替代零散的 `print`
   - 增加了定时清理任务，自动移除超时不活跃玩家
   - 正确清理空房间，节省内存

2. **客户端**
   - 将同步间隔从 50ms 优化到 100ms，减少网络带宽占用
   - 添加了自动重连机制，最多重试 5 次
   - 添加了其他玩家模型清理逻辑，重启游戏不泄漏内存
   - 完善僵尸击晕状态同步，所有客户端一致
   - 修复了重启游戏后UI残留问题

## 技术栈

- **后端**: Python + aiohttp (Websocket + HTTP)
- **前端**: Three.js (WebGL 3D渲染)
- **通信**: WebSocket 实时同步玩家状态
